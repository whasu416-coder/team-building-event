# HTML/CSS 웹 삽입형 숙소 주변 편의시설 약도 제작 프롬프트

아래 프롬프트를 **Antigravity / Cursor / Claude Code / 웹 제작 AI**에 그대로 입력하면 됩니다.  
목표는 이미지 한 장짜리 약도가 아니라, `index.html` 안에 바로 삽입 가능한 **HTML/CSS 웹 섹션형 약도**를 구현하는 것입니다.

---

## 최종 프롬프트

```text
현재 제작 중인 “군산 단합대회 안내 홈페이지”에 삽입할 수 있는 
숙소 주변 편의시설 안내 약도 섹션을 HTML/CSS로 구현해줘.

[구현 목적]
- 제일오투그란데 1단지 306동 정문을 중심으로 도보 10분 이내 편의시설을 안내하는 웹용 약도 섹션 제작
- 구글 지도 캡처를 그대로 사용하지 말고, 위치 관계만 참고해서 새롭게 그린 약도형 UI로 구현
- 실제 지도 API가 아니라, HTML/CSS 기반의 정적인 안내 약도 형태로 제작
- 단합대회 안내 홈페이지에 자연스럽게 들어가는 고급스럽고 깔끔한 대기업 행사 안내자료 스타일로 구현

[중심 위치]
- 중심지는 “제일오투그란데 1단지 306동 정문”
- 306동은 약도 중앙에 가장 크게 표시
- 라벨은 “숙소 · 306동 정문” 또는 “제일오투그란데 1단지 306동 정문”으로 표시
- 숙소 마커는 다른 시설보다 크고 눈에 띄게 디자인

[표시 범위]
- 숙소 기준 도보 10분 이내
- 약도에는 너무 많은 도로와 건물을 넣지 말고, 핵심 도로와 시설만 단순화해서 표시

[반드시 표시할 시설]
1. 롯데마트 군산점
   - 숙소 기준 동쪽 방향에 배치
   - 쇼핑카트 또는 마트 아이콘 사용
   - 예상 도보시간 표시: “도보 약 8~10분”

2. 다이소 군산점
   - 숙소 기준 동쪽 방향, 롯데마트보다 조금 더 가까운 위치로 배치
   - 쇼핑백 또는 생활용품 아이콘 사용
   - 예상 도보시간 표시: “도보 약 6~8분”

3. 가까운 편의점
   - 숙소 주변 도보권 내에 배치
   - 편의점 아이콘 사용
   - 라벨: “가까운 편의점”
   - 예상 도보시간 표시: “도보 약 3~5분”

4. 가까운 약국
   - 숙소 주변 도보권 내에 배치
   - 약국 십자 아이콘 사용
   - 라벨: “가까운 약국”
   - 예상 도보시간 표시: “도보 약 5~8분”

[지도 방향 참고]
- 숙소는 중앙에 배치
- 롯데마트 군산점과 다이소 군산점은 숙소의 오른쪽, 즉 동쪽 방향에 배치
- 주요 도로는 단순화해서 표현
- 수송로, 축동로, 월명로 등 주요 도로명은 필요할 경우 작게 표시
- 북쪽 방향 표시 “N”을 약도 우측 상단에 작게 배치

[디자인 스타일]
- 대기업 행사 안내자료 느낌
- 애플 홈페이지처럼 여백이 넓고 깔끔한 구성
- 흰색 또는 아주 연한 회색 배경
- 카드형 섹션 디자인
- 둥근 모서리, 은은한 그림자, 정돈된 타이포그래피 사용
- 전체적으로 고급스럽고 미니멀하게 구현
- 과한 색상 사용 금지
- 숙소는 네이비 또는 블루 계열로 강조
- 편의시설은 아이콘과 작은 컬러 태그로 구분
- 약도는 복잡한 실제 지도처럼 만들지 말고, 인포그래픽형으로 단순화

[레이아웃]
- 전체 섹션 제목: “숙소 주변 편의시설 안내”
- 부제목: “306동 정문 기준 도보 10분 이내 주요 편의시설”
- 상단에는 제목과 설명
- 중앙에는 약도 영역
- 하단 또는 우측에는 시설 목록 카드 배치
- 시설 목록 카드에는 다음 정보를 표시:
  - 시설명
  - 분류
  - 예상 도보시간
  - 간단한 설명

[시설 목록 예시]
- 롯데마트 군산점 / 대형마트 / 도보 약 8~10분 / 생필품·간식 구매 가능
- 다이소 군산점 / 생활용품 / 도보 약 6~8분 / 생활용품·소모품 구매 가능
- 가까운 편의점 / 편의점 / 도보 약 3~5분 / 간단한 간식·음료 구매
- 가까운 약국 / 약국 / 도보 약 5~8분 / 비상약·상비약 구매

[HTML/CSS 구현 방식]
- 기존 index.html 안에 바로 붙여 넣을 수 있는 섹션 코드로 작성
- HTML, CSS만 사용
- 별도 지도 API 사용하지 않음
- JavaScript는 꼭 필요하지 않으면 사용하지 않음
- 반응형으로 구현
  - PC에서는 약도와 시설 목록이 좌우 또는 넓은 카드형으로 보이게
  - 모바일에서는 약도 아래에 시설 목록이 세로로 정렬되게
- CSS 클래스명은 기존 코드와 충돌하지 않도록 `facility-map-section`, `facility-map-card`, `facility-marker`처럼 명확하게 작성
- 아이콘은 이모지 또는 CSS 기반 아이콘으로 구현 가능
- 외부 라이브러리 없이 동작하게 작성

[약도 구현 방식]
- `.map-canvas` 같은 div를 만들고 그 안에 도로, 숙소, 시설 마커를 absolute position으로 배치
- 도로는 CSS border 또는 background로 단순화된 선 형태로 표현
- 주요 도로는 밝은 회색 라인으로 표시
- 숙소 마커는 중앙에 크게 배치
- 각 편의시설 마커는 위치에 맞게 배치
- 도보 동선은 점선 또는 얇은 곡선 느낌의 라인으로 표현
- 각 마커에는 아이콘, 시설명, 도보시간을 함께 표시

[접근성 및 가독성]
- 텍스트는 모바일에서도 읽기 쉬운 크기 유지
- 색상 대비를 충분히 확보
- 시설명과 도보시간이 겹치지 않게 배치
- 지도 영역이 너무 복잡해 보이지 않도록 여백 확보
- 작은 화면에서는 마커 라벨이 겹치지 않도록 카드 목록을 우선적으로 읽기 좋게 구성

[권장 크기]
- PC 기준 지도 영역: 가로 100%, 최대 너비 1200px
- 약도 비율: 1200x800px 느낌의 가로형
- 섹션 전체는 기존 홈페이지 폭에 맞춰 자연스럽게 들어가도록 구성

[결과물]
- 완성된 HTML 섹션 코드
- 완성된 CSS 코드
- 기존 index.html의 어느 위치에 삽입하면 좋은지도 주석으로 안내
- 기존 디자인과 어울리도록 너무 튀지 않게 제작

[중요 조건]
- 구글 지도 이미지를 그대로 캡처해서 사용하지 말 것
- 위치 관계만 참고한 새 약도형 UI로 만들 것
- 306동이 가장 먼저 눈에 들어오게 할 것
- 롯데마트 군산점, 다이소 군산점, 가까운 편의점, 가까운 약국은 반드시 포함할 것
- 도보시간을 반드시 표시할 것
- 홈페이지 삽입용이므로 반응형과 가독성을 최우선으로 할 것
- 전체 분위기는 “공식 행사 안내자료 + 고급 웹 인포그래픽” 느낌으로 만들 것
```

## 추가 지시문

결과물의 품질을 더 높이고 싶다면 위 프롬프트 마지막에 아래 문장을 추가하세요.

```text
단순한 이미지 생성이 아니라 실제 index.html에 삽입 가능한 HTML/CSS 코드로 작성해줘. 
약도는 SVG 또는 div/CSS 기반으로 직접 그려줘. 
배경 이미지에 의존하지 말고, 도로·마커·라벨·시설카드가 모두 코드로 구현되게 해줘.
```

---

## 추천 구현 방식

가장 추천하는 구현 방식은 **SVG 약도 + HTML 시설 카드** 조합입니다.

| 방식 | 추천도 | 이유 |
|---|---:|---|
| SVG 약도 | 매우 높음 | 선명하고 반응형에 강함 |
| div + absolute 마커 | 높음 | 수정이 쉽고 구현이 빠름 |
| 지도 이미지 삽입 | 낮음 | 모바일 대응과 수정이 어려움 |
| 구글지도 iframe | 낮음 | 행사 안내용 약도 느낌이 약함 |
